## Description

This project is done as a submission for an assignment given to me on the 45th day of Full-Stack Web Development Bootcamp by <a href="https://dibimbing.id">dibimbing.id</a>. This project is done utilizing React.js and Tailwindcss

## Goals

The goal of this project is as follows:

1. Students are able to discern which part of an HTML is to be made into components,
2. Students are capable to apply stylization to a web page using React.js, and
3. Students are able to properly use `props` to send data towards components from parents.
